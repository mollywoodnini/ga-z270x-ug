# Some assorted color presets I have made.

# POLICE (Epilepsy Warning)
liquidctl set logo color pulse ffffff --speed fastest
liquidctl set ring color moving-alternating 0000ff ff0000 --speed fastest

# PALE PURPLE AND FROST
liquidctl set logo color breathing ffffff --speed slower
liquidctl set ring color tai-chi ae27ff 54ffff --speed slower

# GULF RACING
liquidctl set ring color covering-marquee 34f4ff ff700a --speed normal
liquidctl set logo color fixed 0000ff

# PEACH
liquidctl set logo color fixed ff3535
liquidctl set ring color marquee-4 ff3535 --speed slower

# 1984
liquidctl set logo color breathing ffffff --speed slower
liquidctl set ring color fading ae27ff 54ff54 54ffff --speed slower

# COVER ME IN RED
liquidctl set ring color covering-marquee 110000 550000 990000 DD0000 --speed faster
liquidctl set logo color breathing ff0000

# HOT RING (this is esentially a really fine controlled breathing effect where the light doesnt go off)
liquidctl set ring color fading 110000 550000 990000 DD0000

# RAINBOW FADE
liquidctl set ring color fading ff0000 ff9600 ffff00 00ff00 0000ff 9600ff
liquidctl set logo color fixed 555555

# RAINBOW COVER
liquidctl set ring color covering-marquee ff0000 ff9600 ffff00 00ff00 0000ff 9600ff --speed faster
liquidctl set logo color fixed 555555

# DISCO
liquidctl set ring color moving-alternating 111111 aaaaaa --speed slower
liquidctl set logo color fixed ffffff

# FROOT LOOPS
liquidctl set ring color super-wave ff0000 ff9600 ffff00 00ff00 0000ff 9600ff --speed fastest
liquidctl set logo color pulse ffffff --speed faster

# EASY EYES AMBER
liquidctl set ring color fading ff5500 662200 --speed slower
